### react-koa2-ssr 
基于create-react-app 和 koa2 脚手架 搭建的 react 服务端渲染项目骨架

### start

git clone https://github.com/yangfan0095/react-koa2-ssr

npm install 

前端开发 
cd client 
npm run sart

服务端运行

先在客户端执行打包
cd client 

npm run build

进入服务端文件目录
cd server

npm run start
